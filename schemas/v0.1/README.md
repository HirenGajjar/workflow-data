# Schema v0.1

This directory contains the initial Clarvia workflow-data schemas.

The v0.1 schema is intentionally small and conservative.

It is designed to support:

- source-backed workflow objects,
- provenance,
- review states,
- validation,
- static outputs,
- and future exports.

Schemas may change before the first stable release.

Breaking changes should be documented in `CHANGELOG.md`.
